CSS (Estilização)

CSS significa "Cascading Style Sheets". É a linguagem de estilização. Se o HTML é o esqueleto, o CSS é a "pintura", a aparência.

Sintaxe Básica

A estrutura do CSS é sempre seletor { propriedade: valor; }.


        /* 'h1' é o seletor */
        /* 'color' é a propriedade */
        /* 'blue' é o valor */
        h1 {
          color: blue;
          font-size: 24px;
        }

Seletores (Como escolher o que estilizar)

Seletores são o jeito de dizer ao CSS qual elemento HTML você quer mudar.

  • Tag: Afeta todas as tags daquele tipo. (Ex: p, h1, div).
  • ID: Usa o #. É um identificador ÚNICO. (Ex: #menu-principal).
  • Classe: Usa o .. Pode ser usado em VÁRIOS elementos. (Ex: .botao).
]
        /* Pega todas as tags 'p' */
        p {
          line-height: 1.5;
        }

        /* Pega SÓ o elemento com id="logo" */
        #logo {
          width: 150px;
        }

        /* Pega TODOS elementos com class="item-lista" */
        .item-lista {
          font-weight: bold;
        }

As 3 Formas de Usar CSS

  1. CSS Externo (O melhor jeito):

    Criar um arquivo style.css separado e "linkar" ele no <head> do HTML. É o mais organizado.

    <link rel="stylesheet" href="style.css">
  2. CSS Interno:

    Colocar o CSS dentro da tag <style>, direto no <head> do HTML.

  3. CSS Inline:

    Colocar o estilo direto na tag HTML usando o atributo style. (Ex: <p style="color: red;">...</p>). Deve ser evitado.

Layouts

Para organizar a página (ex: colunas, itens lado a lado), os mais usados são Flexbox (display: flex;) e Grid (display: grid;).
