HTML (Introdução)

HTML é o "HyperText Markup Language". Basicamente, é a linguagem de marcação, o esqueleto de toda página web.

Front-end vs. Back-end

  • Front-end: É a parte visual do site, o que o usuário vê e interage. É feito com HTML, CSS e JS.
  • Back-end: Onde fica a regra de negócio. Onde os cadastros e logins são processados.

O que são Tags?

São os "comandos" do HTML. Elas vêm entre < e >. Quase toda tag tem uma de abertura e uma de fechamento.

Exemplo: <p>Isso é um parágrafo.</p>.

O que são Atributos?

São informações extras que a gente passa para uma tag. Elas ficam dentro da tag de abertura.


        <!-- 'src' e 'alt' são atributos da tag 'img' -->
        <img src="foto.jpg" alt="Descrição da foto">

Estrutura Básica de uma Página

Todo arquivo HTML tem essa estrutura:


        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <title>Título que aparece na aba</title>
        </head>
        <body>
        <!-- O conteúdo que aparece no site vai aqui -->
        <h1>Meu site</h1>
        </body>
        </html>

Algumas Tags Importantes

  • <h1>, <h2>... <h6>: Tags de título, para hierarquia.
  • <p>: Parágrafo.
  • <div>: Uma "caixa" genérica para agrupar e organizar conteúdo.
  • <a href="url">...</a>: Para criar links.
  • <img src="arquivo">: Para mostrar imagens.
  • <ul> / <ol> / <li>: Para criar listas (não ordenadas e ordenadas).
