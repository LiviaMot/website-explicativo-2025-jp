JavaScript (Interatividade)

JavaScript (JS) é a linguagem de programação que dá vida e interatividade à página. Se HTML é o esqueleto e CSS é a pintura, o JS é o "motor".

Funções do JS

  • Dar interatividade (cliques, menus, pop-ups).
  • Manipular o DOM (mudar o conteúdo da página sem recarregar).
  • Validar formulários (verificar se o e-mail está certo, etc).
  • Comunicação Assíncrona (buscar dados de um servidor, tipo o clima).

O que é o DOM?

DOM significa "Document Object Model". É a representação do HTML que o JS consegue "ler" e "modificar". Quando o JS muda o DOM, a página muda pro usuário.

Manipulando o DOM

Comandos principais para "mexer" na página:

  • document.getElementById('id-do-elemento'): Pega um elemento pelo seu ID.
  • document.querySelector('.classe-css'): Pega o PRIMEIRO elemento que bater com o seletor CSS (igual ao CSS).
  • document.createElement('div'): Cria um novo elemento (uma div, no caso), mas ele ainda não aparece na tela.
  • elementoPai.appendChild(elementoFilho): Pega o elemento novo e "pendura" ele dentro de outro elemento que já está na tela.

Eventos (Ouvindo o Usuário)

Eventos são como o JS "ouve" o que o usuário faz (clicar, digitar, passar o mouse). O mais usado é o addEventListener.


        // 1. Pega o botão do HTML
        const meuBotao = document.getElementById('botao-comprar');

        // 2. Adiciona um "ouvinte" de clique nele
        meuBotao.addEventListener('click', () => {
          alert('Você clicou no botão!');
        });

Eventos comuns: click, submit (para formulários), mouseover (passar o mouse por cima), keydown (apertar uma tecla).
