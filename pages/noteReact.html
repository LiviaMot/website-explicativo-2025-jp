React

React é uma biblioteca JavaScript para construir interfaces de usuário (UI). A ideia principal dele é a componentização.

O que são Componentes?

É o conceito de quebrar a interface em pedaços pequenos e reutilizáveis. Em vez de uma página gigante, você tem:

  • Um componente <Header />
  • Um componente <Footer />
  • Um componente <CardProduto /> (que você pode reusar várias vezes)

JSX (JavaScript XML)

É uma sintaxe que parece HTML, mas que escrevemos dentro do JavaScript. É o que usamos para definir o que um componente vai mostrar na tela.


        function Botao() {
          // Isso parece HTML, mas é JSX:
          return (
          <button class="meu-botao">Clique Aqui</button>
          );
        }

Props vs. State

  • Props (Propriedades): Informações que um componente "pai" passa para um "filho". São só de leitura.

    Ex: <CardProduto titulo="Produto 1" />. O titulo é uma prop.

  • State (Estado): É a "memória" interna de um componente. Quando o estado muda, o React "redesenha" aquele componente. Usamos o useState. 
    
              // 'contador' é o estado, 'setContador' é a função pra mudar ele
              const [contador, setContador] = useState(0);

Como criar um projeto (Vite)

O professor avisou que o create-react-app (CRA) foi descontinuado. A forma moderna de começar é com o Vite.


        # 1. Cria o projeto
        npm create vite@latest . --template react

        # 2. Instala as dependências
        npm install
        
        # 3. Roda o projeto
        npm run dev

Rotas (Navegação) com React Router

React é uma "Single Page Application" (SPA), ou seja, ele não recarrega a página. Para ter várias páginas (tipo /sobre, /contato), usamos a biblioteca react-router-dom.

Passos para usar o Router:

  1. Instalar: npm i react-router-dom
  2. No main.jsx: Envolver o <App /> com o <BrowserRouter>.
  3. No App.jsx: Configurar as rotas: 
    
              import { Routes, Route, Link } from 'react-router-dom';
              import Home from './pages/Home'; // sua página Home
              import Sobre from './pages/Sobre'; // sua página Sobre
              
              function App() {
                return (
                  <>
                  <nav>
                  {/* Link é o substituto do <a> */}
                  <Link to="/">Home</Link>
                  <Link to="/sobre">Sobre</Link>
                  </nav>

                {/* Define quais componentes aparecem em quais URLs */}
                <Routes>
                  <Route path="/" element={<Home />} />
                  <Route path="/sobre" element={<Sobre />} />
                </Routes>
              </>
              )
            }
